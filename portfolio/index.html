    Tamil Flash cards

    This is a tool for learning Tamil with its correct pronounciation. I recorded my voices and provided images for easy recognition. It is possible to auto-play the flash cards. The flash cards are grouped under sets such as alphabets, fruits, animals, etc. The audio plays for each flash card automatically when it is displayed. There is an option to mute. It is also possible to manually navigate between cards in a set apart from auto play.
